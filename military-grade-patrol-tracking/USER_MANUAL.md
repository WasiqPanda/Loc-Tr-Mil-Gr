# Military Grade Patrol Tracking - User Manual

## Overview
Military Grade Patrol Tracking (MGPT) is a distributed command and control ecosystem designed for real-time patrol operations. It provides situational awareness, unit tracking, incident reporting, and simulated communication capabilities.

## Getting Started

### 1. Accessing the System
*   **URL:** Navigate to the provided application URL.
*   **Login:**
    *   Click "INITIATE SECURE HANDSHAKE".
    *   The system will authenticate your session.
*   **Trial:** New users receive a 48-hour full-access trial.

### 2. Dashboard Interface
The interface is divided into three main sections:
*   **Header:** Displays system status, time (Zulu), and quick actions (Report, Admin).
*   **Map View (Center):** Real-time tactical map showing units, incidents, and trails.
*   **Sidebar (Left):** List of active units. Can be collapsed for a larger map view.
*   **Intel Panel (Right):** Details for selected units or incidents.

## Features

### Real-Time Tracking
*   **Units:** Active units appear as icons on the map.
*   **Trails:** Movement history is visualized as a line.
    *   **Start Point:** Marked with a small dot.
    *   **Current Position:** Marked with the unit icon.
    *   **Selection:** Click a unit to highlight its trail and view details.
*   **Track My Device:**
    *   Click the **"TRACK DEVICE"** button in the header.
    *   Allow geolocation permissions.
    *   Your device will appear on the map as "THIS DEVICE" and start leaving a trail.

### Incident Management
*   **Reporting:**
    *   Click **"REPORT"** in the header.
    *   Select Type (Hostile, Medical, etc.) and Priority.
    *   Enter a description.
    *   The incident will appear on the map and in the feed.
*   **Managing:**
    *   Click an incident on the map.
    *   **Dispatch:** Assign units (simulated).
    *   **Resolve:** Mark as resolved.
    *   **Edit/Delete:** Update or remove the record.

### Communications (Premium)
*   **Comms:** Select a unit and click **"COMMS"** to open the secure channel.
*   **Features:** Timer, Mute, End Call.
*   *Note: Requires Premium Tier.*

*   **Location Not Updating:** Ensure GPS is enabled and permissions are granted.
*   **Map Blank:** Check internet connection.
*   **System Locked:** Your trial may have expired. Use the Admin Console to extend or upgrade.
